# Practica 1: Creacion de clases y encapsulamiento en PHP

## Objetivo: Aplicar los fundamentos de la programacion Orientada a Objetos en PHP
    implementando una clase con atributos privados, constructor y metodos de acceso 
    siguiendo buenas practicas de encapsulamiento. 

  ##Clase creada 
    Como atributo privado se creo "$nombre" y "$correo" para que aseguremos el encapsulamiento 

  ## Constructor 
    Este nos ayuda a recibir los parametros de $nombre y $correo

  ## Metodos publicos
    Como metodos publicos se crearon "getCorreo()" "setNombrre()" "setCorreo()"y son los que interactuan 
    en manera mas segura en nuestro codigo
